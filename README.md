# 📋 ToDo App (React + LocalStorage)

Простое ToDo-приложение, созданное на **React** с использованием функциональных компонентов, хуков **useState** и **useEffect**.  
Приложение позволяет добавлять задачи, удалять их, очищать весь список и сохранять данные в **localStorage**, чтобы задачи не исчезали при перезагрузке страницы.

---

## 🚀 Функциональность

- ✨ Добавление задач  
- 🗑 Удаление отдельной задачи  
- 🔥 Кнопка «Очистить всё»  
- 💾 Автосохранение данных в localStorage  
- ⚛️ Компонентная структура:
  - App
  - AddTaskForm
  - ToDoList
  - ToDoItem
